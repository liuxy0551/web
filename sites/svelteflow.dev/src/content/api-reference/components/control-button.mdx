# &lt;ControlButton />

[Source on GitHub](https://github.com/xyflow/xyflow/blob/main/packages/svelte/src/lib/plugins/Controls/ControlButton.svelte)

The `<ControlButton />` component is used to render custom/ additional buttons
for the `<Controls />` component.

```svelte
<script lang="ts">
  import { Controls, ControlButton } from '@xyflow/svelte';
</script>

<Controls>
  <ControlButton onclick={() => console.log('⚡️')}>
    ⚡️
  </ControlButton>
</Controls>
```
