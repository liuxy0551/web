### Minor Changes
-   [#5299](https://github.com/xyflow/xyflow/pull/5299) Thanks [@printerscanner](https://github.com/printerscanner)! - Add `ariaRole` prop to nodes and edges
-   [#5280](https://github.com/xyflow/xyflow/pull/5280) Improve typing for Nodes
-   [#5276](https://github.com/xyflow/xyflow/pull/5276) Add an `ease` and `interpolate` option to all function that alter the viewport
-   [#5277](https://github.com/xyflow/xyflow/pull/5277) Thanks [@printerscanner](https://github.com/printerscanner)! - Add `ariaLabelConfig` prop for customizing UI text like aria labels and descriptions.
-   [#5317](https://github.com/xyflow/xyflow/pull/5317) Add `domAttributes` option for nodes and edges
-   [#5326](https://github.com/xyflow/xyflow/pull/5326) Prevent NodeResizer controls to become too small when zooming out
-   [#5308](https://github.com/xyflow/xyflow/pull/5308) Thanks [@printerscanner](https://github.com/printerscanner)! - Focus nodes on tab if not within the viewport and add a new prop `autoPanOnNodeFocus`
