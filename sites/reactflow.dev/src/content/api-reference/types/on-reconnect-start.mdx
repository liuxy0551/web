# OnReconnectStart

[Source on GitHub](https://github.com/xyflow/xyflow/blob/main/packages/system/src/types/general.ts#L84)

The `OnReconnectStart` type represents a callback function that is called when starting to reconnect an existing edge. It receives the mouse or touch event, the edge being reconnected, and the type of handle being used.

```ts
type OnReconnectStart<EdgeType extends EdgeBase = EdgeBase> = (
  event: MouseEvent | TouchEvent,
  edge: EdgeType,
  handleType: HandleType,
) => void;
```
